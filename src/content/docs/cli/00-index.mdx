## Key Features

- **Intelligent Version Management**: Supports major, minor, patch, prerelease, branch-based, and calendar versioning strategies
- **Multi-Registry Support**: Publish to multiple NPM registries, GitHub Releases, and GitLab Releases
- **Branch-Specific Configuration**: Different publishing strategies for main, develop, feature, and hotfix branches
- **Git Integration**: Automatic tag creation, commit verification, and push operations
- **CI/CD Ready**: Works seamlessly with GitHub Actions, GitLab CI, Jenkins, and other CI/CD platforms
- **Extensible Hooks**: Pre/post publish and version hooks for custom automation
- **Dry Run Mode**: Test your configuration without making actual changes

## Quick Start

Get started with AppSwap CLI in just a few steps:

1. **Install the CLI**
   ```bash
   npm install -g @appswap/cli
   ```

2. **Initialize your project**
   ```bash
   appswap init
   ```

3. **Configure your publishing strategy**
   ```bash
   # Edit .appswap/publish.yaml
   # Edit .appswap/version.yaml
   ```

4. **Publish your package**
   ```bash
   appswap publish
   ```
